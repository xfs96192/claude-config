# 产品运作概览字段说明

## 核心字段

| 字段名 | 说明 | 数据类型 | 示例 |
|-------|------|---------|------|
| 产品代码 | 产品唯一标识 | string | 9K251690 |
| 产品简称 | 产品名称 | string | 兴银理财丰利合享1号 |
| 母产品代码 | 母产品代码(子产品关联用) | string | 9K251690 |
| 净资产(亿) | 产品规模 | float | 10.52 |
| 单位净值 | 当前单位净值 | float | 1.0234 |
| 累计年化 | 成立以来年化收益率 | string/float | 3.25% |
| 本期年化 | 当期年化收益率 | string/float | 3.10% |
| 杠杆 | 杠杆倍数 | float | 1.15 |
| 组合久期 | 债券组合久期(年) | float | 0.85 |
| 组合静态 | 组合静态收益率 | float | 0.028 |
| 权益仓位 | 权益资产占比 | float | 0.05 |
| 衍生品仓位 | 衍生品资产占比 | float | 0.02 |
| 运作天 | 已运作天数 | int | 365 |
| 剩余天 | 距到期剩余天数 | int | 180 |
| 产品经理 | 投资经理(多人用逗号分隔) | string | 朱轶伦,任雁 |
| 起息日 | 产品成立日 | date | 2024-01-15 |
| 到期日 | 产品到期日 | date | 2026-01-15 |
| 业绩基准 | 业绩比较基准 | string | 3.0%-3.5% |
| 销售费 | 年化销售费率 | float | 0.003 |
| 管理费 | 年化管理费率 | float | 0.003 |
| 托管费 | 年化托管费率 | float | 0.0005 |
| 币种 | 产品币种 | string | 人民币/美元 |

## 分类字段

| 字段名 | 说明 | 取值范围 |
|-------|------|---------|
| 形态 | 产品运作形态 | 封闭、定开、日开、客户周期 |
| 多资产品类 | 部门产品分类 | 非标驱动-丰利合享1年R2、无权系列-悦动稳享R2等 |
| 产品部品类 | 公司产品分类 | - |
| 母分类 | 二级分类 | 悦动稳享短持、悦动稳/纯享、和瑞BOF等 |
| 总分类 | 一级分类 | 中低波稳健产品、中高波市值产品、外币产品 |
| 母子标志 | 母子产品标识 | 非母子产品、母产品、子产品 |
| 主要展示 | 是否主要展示份额 | 是、否 |
| 是否招商 | 招商渠道标签 | 招商独有、招商共有、非招商 |
| 渠道明细 | 代销渠道列表 | 兴业,招商,中信 |

## 持仓盈亏明细字段

| 字段名 | 说明 | 用途 |
|-------|------|------|
| 产品代码 | 持有该资产的产品代码 | 关联产品 |
| 债券简称/资产简称 | 资产名称 | 资产识别 |
| 外部代码 | Wind/Bloomberg代码 | 获取市场数据 |
| 新版资产小类 | 资产细分类型 | 资产分类统计 |
| 日终市值(元)-产品法估值 | **规模统计首选字段** | 持仓规模 |
| 持仓成本(元) | 资产成本 | 盈亏计算 |
| 持有期收益 | 持有期间收益 | 业绩归因 |
